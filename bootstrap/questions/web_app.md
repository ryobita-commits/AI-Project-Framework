# Initial Questions: Web App

1. What product are you trying to build?
2. Who are the target users?
3. What is the main workflow the user should complete?
4. What is the smallest usable first release?
5. Which platform constraints matter most?
6. What frontend and backend assumptions already exist?
7. Does the project require authentication, payments, or external APIs?
8. What is out of scope for the first release?
9. What metrics define success?
10. What decisions are still open?
