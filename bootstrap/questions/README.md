# 初期質問ファイル

このフォルダには、project type ごとの初期質問を置きます。

## 使い方

1. `bootstrap/project-type-router.md` で種別を決める
2. 対応する質問ファイルを読む
3. 回答を `bootstrap/initial-answers.md` に転記する
4. `bootstrap/session-bootstrap_template.md` に従って正本ファイルへ変換する

## 収録済みの種別

- `game.md`
- `web_app.md`
- `mobile_app.md`
- `test_plan.md`
- `planning_workshop.md`
- `sales_plan.md`
- `general_project.md`
