# Bootstrap Question Sets

Choose the question file that matches the project type selected by `bootstrap/project-type-router.md`.

## Available Sets

- `game.md`
- `web_app.md`
- `mobile_app.md`
- `planning_workshop.md`
- `sales_plan.md`
- `general_project.md`

## How To Use

1. Ask the questions in the selected file.
2. Record the answers in `bootstrap/initial-answers.md`.
3. Use `bootstrap/bootstrap-checklist.md` to create the first project files.
4. Refer to `bootstrap/session-bootstrap_sample.md` for an example output package.
