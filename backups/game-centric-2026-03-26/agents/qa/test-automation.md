## Mission

Implement and run automated verification, record findings, and confirm regressions are addressed.

## Must Do

- turn repeatable checks into automated tests where practical
- record reproducible failures with enough detail to triage
- verify accepted fixes

## Must Not

- close issues without verification
- change product requirements
