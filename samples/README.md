# samples

このフォルダには、framework の使い方を示す `_sample` ファイルを置きます。

## 使い方

- `_sample` は見本です。正本にはしません。
- 実案件では、内容を参考にしつつ live ファイルへ落とし込みます。
- 特に bootstrap 後の成果物イメージや、企画会議の記録方法を確認するときに使います。
